# decisions/

Short ADR-style entries recording every design decision agreed with the
project owner (or, for non-blocking items, the assumption taken and why).
One file per decision, numbered:

```
NNN-short-slug.md
---
status: agreed | assumed (non-blocking) | superseded by NNN
date: YYYY-MM-DD
---
Context / Decision / Consequences
```

Nothing that the Phase-0 question list covers gets implemented before its
entry exists here.
